# 미니인턴 과제 - 바우디움

SwiftUI를 사용해서 일정을 관리할 수 있는 달력을 구현

## **구현 기능**

- [x]  **구현한 필수 기능**
- 이벤트 스택 : 한 날짜에 이벤트가 2개 이상 존재할 때의 상황
    - 하루를 4분할 하여 이벤트가 있음을 표시
    - 이벤트가 없는 날짜는 표시하지 않음 
- [ ]  **구현하지 못한 기능**
- 캘린더의 날짜 클릭 시 해당 날짜가 눌렸는 표시하는 기능
- 캘린더 무한 스크롤 기능
